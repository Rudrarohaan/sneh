# Valentine Proposal Website

A single-page interactive Valentine proposal with a smooth, glassmorphism design.

## How to Run

1.  Simply open the `index.html` file in your preferred web browser.
2.  The animations and interactions will work immediately.
3.  Works best on modern browsers (Chrome, Firefox, Edge, Safari).

## Features

*   **Responsive Design**: Looks good on both desktop and mobile.
*   **Touch Support**: "NO" button moves on touch as well as hover/click.
*   **Smooth Animations**: CSS transitions for button movement and content fading.
*   **Glassmorphism Effect**: Backdrop blur and semi-transparent layers.

## Customization

*   **Text**: Edit the text in `index.html` to personalize the message.
*   **Colors**: Modify the gradients in `style.css` if you prefer a different palette.
